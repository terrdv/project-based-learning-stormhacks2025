# Polaris 

## Tecb Stack
- React + Vite (TypeScript)
- Node.js and Express
- Supabase (Auth and Database)
- Gemini API 

## Setup Backend


```sh
cd server
npx nodemon index.js

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Setup Frontend


```sh
cd client
npm run dev

```
